# 项目概述

## 项目基本信息
- **项目名称**: 食道超声模拟软件
- **项目目标**: 开发基于Web技术的跨平台食道超声模拟软件，主要运行在Android平板设备上
- **核心功能**: 3D心脏模型可视化、食道路径模拟、实时超声切面计算
- **技术栈**: React + TypeScript + Three.js + Capacitor
- **部署模式**: 纯客户端离线运行

## 核心需求
1. **多平台支持**: Android平板为主，支持桌面和Web端
2. **离线运行**: 无需网络连接，保护医疗数据隐私
3. **实时交互**: 触摸控制探头位置和角度，实时计算切面
4. **医学准确性**: 基于真实3D心脏模型和食道路径数据
5. **开发效率**: 使用VS Code + Cline + LLM辅助开发

## 项目状态
- **当前阶段**: 方案设计完成，准备开始开发
- **已完成工作**: 技术方案文档、开发环境配置指南、实施路线图
- **下一步**: 搭建开发环境，开始基础框架开发

## 关键决策
1. **技术选型**: Web技术栈而非Unity，便于VS Code + Cline开发
2. **部署架构**: 纯客户端而非服务端，满足医疗数据安全要求
3. **开发工具**: 充分利用LLM辅助开发提高效率
